# Arduino-Rock-Scissors-Paper-Ax12
Big arduino rock scissors paper machine with AX-12 motor

## 아두이노 가위바위보 머신 ver.AX12
- AX-12A 모터 3개
- 220V to 12V 어댑터 1개
- 아두이노 우노 1개
- 적외선 거리센서 1개

## Pin 
적외선 거리센서 - 3V3, GND, A0
Ax-12A - digital 8

### AX-12A ID
SCISSOR 1
ROCK 2
PAPER 14

### AX-12A mode
Joint mode

### Baud Rate
57600bps
